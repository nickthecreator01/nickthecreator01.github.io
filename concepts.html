Key Conecpts & Defintion

This section helps define and explains the key concepts that play important roles in understanding more about web servers and their way of operating

Untangling the Web

Web Server
A computer system that delivers web content to end users via the internet using the HTTP protocol.
HTTP
HyperText Transfer Protocol: the protocol used for transferring data over the internet.
HTTP/HTTPS
HyperText Transfer Protocol: the protocol used for transferring data over the internet securely.
Data Center
A facility that houses computer servers and networking equipment for storing and processing data.
Bandwidth
The maximum rate of data transfer across a given path, typically measured in bits per second.
Latency
The delay between a request being sent and a response being received.
Load Balancing
The process of distributing network traffic across multiple servers to optimize performance.
