Within the topic of web servers, there are infrastructure providers that provide hardware, network connections, and data centers where web servers physically reside. Organizations in this category include Amazon Web Services, the leading cloud provider offering computing and web server hosting, and Google Cloud Platform, a major hyperscale cloud provider responsible for hosting major global server networks. In history web servers have had a major impact on the global economy by being a way to allow businesses, governments, and all types of organizations which allow online services to online users around the world.
All websites and web applications depend of the use of web servers to be able to process requests and deliver different content that are responsible for forming the foundation of important industries. Industries that are in this category are online banking, digital media, cloud computing, and social networking. Web servers also play a major role in the expansion of the rapidly growing cloud computing market, and with that a lot of organizations now host a good number of websites and applications on cloud platforms.

For this research report, I’ve decided to cover a web-related component, namely web servers. A web server is a computer system that delivers web content to end users via a web browser over the internet.
Web servers are software that help users and websites communicate by gathering information. Web servers receive requests from web browsers and, in return, serve web pages, images, videos, and other resources related to the request using HTTP or HTTPS. When a person goes to a website, the web server makes sure that the information typed in the search is accurately related to what’s shown on the web page. The main role of a web server is to properly store, process, and deliver website content to users, acting as a middleman between users and the internet. Web browsers work hand in hand with web servers to help users find what they need.
