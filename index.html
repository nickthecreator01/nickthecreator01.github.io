For this research report, I’ve decided to cover a web-related component, namely web servers. A web server is a computer system that delivers web content to end users via a web browser over the internet. Web servers are software that help users and websites communicate by gathering information. Web servers receive requests from web browsers and, in return, serve web pages, images, videos, and other resources related to the request using HTTP or HTTPS. When a person goes to a website, the web server makes sure that the information typed in the search is accurately related to what’s shown on the web page. The main role of a web server is to properly store, process, and deliver website content to users, acting as a middleman between users and the internet. Web browsers work hand in hand with web servers to help users find what they need.

Web browsers like Chrome or Internet Explorer locate the web server where your website data is stored, then analyze the request and process the information. The way web servers operate is a form of call-and-response. The initial step involves the client sending a request to the server; during this step, the browser locates the server's IP address, establishes a reliable connection, and sends an HTTP request.
